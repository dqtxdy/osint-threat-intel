# UI Sources And Licenses

The final React dashboard uses an original implementation tailored for this capstone project.

Design inspiration:

- OpenCTI: CTI data organization, entity-centric workflows, graph-first thinking.
- IntelOwl: enrichment and analyst workflow patterns.
- TailwindAdmin: MIT-licensed dashboard layout inspiration.
- Shadboard: MIT-licensed Shadcn-style dashboard inspiration.

No proprietary UI assets are copied. The project uses common dashboard patterns such as side navigation, KPI cards, evidence panels, tables, and graph views.

Primary frontend libraries:

- React
- Vite
- Tailwind CSS
- lucide-react
- Recharts
